# 全栈技术博客 - 产品规范
## 项目定位
个人技术博客，支持线上后台编辑文章，数据持久化，公网可正常访问。

## 目标用户
普通访客（浏览文章）、管理员（在线增删改查文章）

## 核心功能
### 前台（公开访问）
1. 首页：文章列表，展示标题、摘要、发布时间、标签
2. 文章详情页：Markdown 渲染、代码高亮、文章目录、代码复制
3. 分类页、标签页：按分类/标签筛选文章
4. 全局：浅色/暗黑主题切换、移动端响应式布局

### 后台管理（需登录鉴权）
1. 管理员账号密码登录，Token 保持登录状态
2. 在线 Markdown 编辑器，左右分栏：左侧编辑、右侧实时预览
3. 文章管理：新增、编辑、删除、上下架文章
4. 基础数据统计展示

## 交互要求
1. 接口请求、操作按钮增加加载状态提示
2. 操作完成给出成功/失败文字反馈
3. 全页面兼容桌面端、手机端
